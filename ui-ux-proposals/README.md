# FinStack AI – Interactive UI/UX Design Proposals

This directory contains **two distinct, fully interactive UI/UX designs** created for the **FinStack AI platform**.

---

## 🎨 Design 1: "Dark Nebula"
📁 **File:** `finstack_design1_dark_nebula.html`

*   **Vibe:** Premium fintech (cyberpunk details meets advanced trading system).
*   **Color Palette:** Deep Space Black background (`#060811`), glowing Emerald Green (`#10b981`), and Electric Blue (`#3b82f6`).
*   **Visual Highlights:** Animated particle canvas background, floating glowing widgets, smooth gradients.

---

## 🎨 Design 2: "Solar Bento"
📁 **File:** `finstack_design2_solar_bento.html`

*   **Vibe:** Modern Gen-Z friendly layout (sleek grid system, vibrant gradients, clean spaces).
*   **Color Palette:** Warm Sand/Cream background (`#f5f0eb`), hot Coral Red (`#ff6b6b`), and Royal Violet (`#7c3aed`).
*   **Visual Highlights:** Animated background gradient vector blobs, 12-column CSS Bento Grid architecture, radial score ring details.

## ⚙️ How to Review and Share
1.  **Open locally:** Simply **double-click** either file to open it in Google Chrome, Safari, Firefox, or Edge. You don't need Node, npm, or any server running.
2.  **Toggle Views:** In the bottom-right corner of each file, there's a floating toggle button to switch between:
    *   `🌐 Landing Page` (Marketing, features list, metrics)
    *   `📊 App Dashboard` (Graphs, portfolio tracker, AI helper chatbot, goals)
3.  **Share with team:** You can send these HTML files directly to your team via Slack, Discord, Email, or host them on a quick hosting link (e.g., Vercel, Netlify, or GitHub Pages).
